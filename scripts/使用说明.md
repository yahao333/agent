### 使用说明

1. **首次使用** - 先用 HITL 模式学习和优化:
   ```bash
   chmod +x ralph.sh ralph-once.sh afk-ralph.sh
   ./ralph-once.sh
   ```

2. **提示成熟后** - 切换到 AFK 模式:
   ```bash
   ./afk-ralph.sh 30
   ```

3. **自定义** - 通过环境变量改变任务来源:
   ```bash
   PLAN_FILE=my-features.json PROGRESS_FILE=session1.txt ./ralph.sh 10
   ```
